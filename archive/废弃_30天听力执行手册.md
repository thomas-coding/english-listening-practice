## 5. 单材料操作动作（每次都一样）

### Daily
1. 首听 1 遍（不暂停），写 1 句主旨 + 4 个关键词。
2. 二听（可暂停），填 8 格细节表。
3. 对照文本纠错，记录至少 3 个“听错 -> 正确”。
4. 口头复述 45-60 秒。

### EnglishPod
1. 先听 `pb`（不暂停），写主旨 + 关键词。
2. 听 `rv` 校正逻辑；听 `dg` 提取表达。
3. 对照文本补漏，记录至少 3 个听错点。
4. 输出 8-10 条可复用口语块。
5. 角色复述 45-60 秒。

### 初来乍到 / YouTube
1. 无字幕首听，写主线（人物/冲突/结果）。
2. 开英文字幕二听，补细节。
3. 回到无字幕复听片段，做 45-60 秒复述。

## 6. 通过标准（是否可以进入下一天）

### Daily 通过
- 有主旨句 + 4关键词。
- 8格细节命中 >= 6。
- 纠错 >= 3 条。
- 复述 >= 45 秒。

### EnglishPod 通过
- `pb`首听主旨成立。
- 8格细节命中 >= 6。
- 口语块 >= 8 条。
- 复述 >= 45 秒。

### 初来乍到 / YouTube 通过
- 无字幕能说主线。
- 二听后 8格命中 >= 5。
- 纠错 >= 3 条。
- 无字幕复述 >= 45 秒。

## 7. 当天判定规则（可控）
- 绿色：全部模块通过，次日正常推进。
- 黄色：1个模块未通过，次日先做 15 分钟补救再上新。
- 红色：2个及以上未通过，次日只做补救，不加新初来乍到/YouTube。
