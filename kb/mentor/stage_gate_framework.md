# Stage-Gate Framework (Listening Progression)

## Stage S0: Controlled decoding
- Main materials: Daily + EnglishPod.
- Promotion criteria (meet all):
  - 7-lesson avg score >= 68
  - first-pass gist >= 60%
  - recap quality >= 2.5

## Stage S1: Semi-authentic bridge
- Main materials: add curated sitcom clips.
- Promotion criteria:
  - 7-lesson avg score >= 74
  - no-sub gist on short clips >= 55%
  - error recurrence rate decreases for 2 consecutive weeks

## Stage S2: Authentic transition
- Main materials: sitcom + short YouTube clips.
- Promotion criteria:
  - 7-lesson avg score >= 78
  - no-sub robustness >= 65
  - recap quality >= 3.5

## Stage S3: YouTube robustness
- Main materials: normal YouTube videos without subtitles (first pass).
- Target criteria:
  - no-sub robustness >= 75 on 8-12 minute videos
  - stable monthly benchmarks (no regression alert)

## Demotion rule
- If regression alert remains true after 3 recovery lessons, demote one stage temporarily.
