# KB Health Score Method

## Purpose
- Quantify whether the mentor system remains reliable and improving.

## Score range
- 0 to 100.

## Components
1. Data integrity (30): required lesson artifacts completeness.
2. Benchmark freshness (20): monthly benchmark completed on time.
3. Personalization coverage (20): mastery trackers actively populated and updated.
4. Regression handling (15): alerts resolved within defined recovery window.
5. Operational readiness (15): playback workflow and bootstrap files usable.

## Interpretation
- >=85: strong
- 70-84: usable but needs targeted maintenance
- <70: at risk, run maintenance before next lesson plan escalation
